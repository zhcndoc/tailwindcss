### 使用逻辑属性 `scroll-pbs` 和 `scroll-pbe`

使用 `scroll-pbs-<number>` 和 `scroll-pbe-<number>` 工具类设置 `scroll-padding-block-start` 和 `scroll-padding-block-end` 逻辑属性，这两个属性会根据书写模式映射到顶部或底部：

```html
<!-- [!code classes:scroll-pbs-6] -->
<div class="snap-y scroll-pbs-6 ...">
  <!-- ... -->
</div>
```

### 使用负值

要使用负的滚动内边距值，请用短横线作为前缀将类名转换为负值：

```html
<!-- [!code classes:-scroll-ps-6] -->
<div class="-scroll-ps-6 snap-x ...">
  <!-- ... -->
</div>
```
