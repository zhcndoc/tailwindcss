## 示例

### 基本示例

使用 `zoom-<number>` 工具类，如 `zoom-75` 和 `zoom-125`，通过 CSS `zoom` 属性缩放元素：

```html
<!-- [!code classes:zoom-75,zoom-100,zoom-125] -->
<div class="zoom-75 ...">
  <!-- ... -->
</div>
<div class="zoom-100 ...">
  <!-- ... -->
</div>
<div class="zoom-125 ...">
  <!-- ... -->
</div>
```
