## 示例

### 允许文本换行

使用 `text-wrap` 工具类将在文本的逻辑点上将溢出的文本换到多行：

<Figure>

<Example padding={false}>
  {
    <div className="mx-auto grid max-w-xs gap-4 border-x border-x-pink-400/30 py-8 text-gray-700 dark:text-gray-400">
      <div className="text-xl font-semibold text-gray-900 dark:text-white">备受喜爱的曼哈顿汤摊关闭</div>
      <p className="text-sm/6">
        今年，纽约人在寒冷的冬季面临更少的温暖，因为这座城市最受尊敬的汤摊意外关闭，接连发生的事件让社区感到困惑。
      </p>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-wrap] -->
<article class="text-wrap">
  <h3>备受喜爱的曼哈顿汤摊关闭</h3>
  <p>纽约人在寒冷的冬季面临...</p>
</article>
```

</Figure>

### 防止文本换行

使用 `text-nowrap` 工具类来防止文本换行，如有必要，允许其溢出：

          备受喜爱的曼哈顿汤摊关闭
        </div>
        <p className="text-sm/6 text-nowrap">
          今年，纽约人在寒冷的冬季面临更少的温暖，因为这座城市最受尊敬的汤摊意外关闭，接连发生的事件让社区感到困惑。
        </p>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-nowrap] -->
<article class="text-nowrap">
  <h3>备受喜爱的曼哈顿汤摊关闭</h3>
  <p>纽约人在寒冷的冬季面临...</p>
</article>
```

</Figure>

### 平衡文本换行

使用 `text-balance` 工具类在每行之间均匀分配文本：

<Figure>

<Example padding={false}>
  {
    <div className="mx-auto grid max-w-xs gap-4 border-x border-x-pink-400/30 py-8 text-gray-700 dark:text-gray-400">
      <div className="text-xl font-semibold text-balance text-gray-900 dark:text-white">
        备受喜爱的曼哈顿汤摊关闭
      </div>
      <p className="text-sm/6">
        今年，纽约人在寒冷的冬季面临更少的温暖，因为这座城市最受尊敬的汤摊意外关闭，接连发生的事件让社区感到困惑。
      </p>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-balance] -->
<article>
  <h3 class="text-balance">备受喜爱的曼哈顿汤摊关闭</h3>
  <p>纽约人在寒冷的冬季面临...</p>
</article>
```

</Figure>

出于性能原因，浏览器将文本平衡限制在约 6 行或更少的块内，因此最适合用于标题。

### 美观文本换行

使用 `text-pretty` 工具可以在牺牲速度的情况下优先考虑更好的文本换行和布局。不同浏览器的行为有所不同，但通常涉及防止孤立词（单独一行的单个词）出现在文本块的末尾。

      </p>
    </div>
  }
</Example>

