## 示例

### 使用默认滚动条宽度

使用 `scrollbar-auto` 工具类来使用浏览器的默认滚动条宽度：

```html
<!-- [!code classes:scrollbar-auto] -->
<div class="scrollbar-auto overflow-auto ...">
  <!-- ... -->
</div>
```

### 隐藏滚动条

使用 `scrollbar-none` 工具类来隐藏滚动条，同时仍然允许元素滚动：

```html
<!-- [!code classes:scrollbar-none] -->
<div class="scrollbar-none overflow-auto ...">
  <!-- ... -->
</div>
```

这些工具类仅支持浏览器关键字 `auto`、`thin` 和 `none`。
