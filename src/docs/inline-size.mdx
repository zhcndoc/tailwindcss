## 示例

### 基础示例

使用 `inline-<number>` 工具类，如 `inline-24` 和 `inline-64`，根据间距比例设置元素的固定内联尺寸：

        <div className="hidden rounded-lg bg-blue-500 px-4 py-2 inline-80 sm:block">inline-80</div>
        <div className="hidden rounded-lg bg-blue-500 px-4 py-2 inline-64 sm:block">inline-64</div>
        <div className="rounded-lg bg-blue-500 px-4 py-2 inline-48">inline-48</div>
        <div className="rounded-lg bg-blue-500 px-4 py-2 inline-40">inline-40</div>
        <div className="rounded-lg bg-blue-500 px-4 py-2 inline-32">inline-32</div>
      </div>
    </div>
  }
</Example>

### 匹配视口宽度

使用 `inline-screen` 工具类让元素跨满整个视口的内联尺寸：

```html
<!-- [!code classes:inline-screen] -->
<div class="inline-screen">
  <!-- ... -->
</div>
```

### 重置内联尺寸

使用 `inline-auto` 工具类在特定条件（如某个响应式断点）下移除元素设置的内联尺寸：

```html
<!-- [!code classes:inline-full,inline-auto] -->
<div class="inline-full md:inline-auto">
  <!-- ... -->
</div>
```
