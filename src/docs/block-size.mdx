### 匹配视口大小

使用 `block-screen` 工具类，使元素块大小等于视口的整个块尺寸：

```html
<!-- [!code classes:block-screen] -->
<div class="block-screen">
  <!-- ... -->
</div>
```

### 匹配动态视口

使用 `block-dvh` 工具类，使元素块大小等于随着浏览器 UI 展开或收缩而变化的视口块尺寸：

```html
<!-- [!code classes:block-dvh] -->
<div class="block-dvh">
  <!-- ... -->
</div>
```

### 匹配最大视口

使用 `block-lvh` 工具类，将元素的块大小设置为视口可能的最大块尺寸：

```html
<!-- [!code classes:block-lvh] -->
<div class="block-lvh">
  <!-- ... -->
</div>
```

### 匹配最小视口

使用 `block-svh` 工具类，将元素的块大小设置为视口可能的最小块尺寸：

```html
<!-- [!code classes:block-svh] -->
<div class="block-svh">
  <!-- ... -->
</div>
```
