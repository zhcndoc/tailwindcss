## 自定义主题

<CustomizingYourTheme
  utility="font"
  name="字体系列"
  customName="display"
  customValue='"Oswald", sans-serif'
>

您还可以为字体系列提供默认的 `font-feature-settings` 和 `font-variation-settings` 值：

```css
@theme {
  --font-display: "Oswald", sans-serif;
  --font-display--font-feature-settings: "cv02", "cv03", "cv04", "cv11"; /* [!code highlight] */
  --font-display--font-variation-settings: "opsz" 32; /* [!code highlight] */
}
```

如果需要，使用 [@font-face](https://developer.mozilla.org/en-US/docs/Web/CSS/@font-face) at-rule 来加载自定义字体：

```css
@font-face {
  font-family: Oswald;
  font-style: normal;
  font-weight: 200 700;
  font-display: swap;
  src: url("/fonts/Oswald.woff2") format("woff2");
}
```

如果从类似 [Google Fonts](https://fonts.google.com/) 的服务加载字体，请确保将 `@import` 置于 CSS 文件的最顶部：

```css
@import url("https://fonts.googleapis.com/css2?family=Roboto&display=swap"); /* [!code highlight] */
@import "tailwindcss";

@theme {
  --font-roboto: "Roboto", sans-serif; /* [!code highlight] */
}
```

浏览器要求 `@import` 语句必须在其他规则之前，因此 URL 引入需要在诸如 `@import "tailwindcss"` 之类的内联作用的引入之上。
